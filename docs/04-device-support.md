
# Supported devices

balenaSound has been tested to work on the following devices:

| Device Type  | Multi-room disabled | Multi-room server | Multi-room client |
| ------------- | ------------- | ------------- | ------------- |
| Raspberry Pi (v1 / Zero / Zero W) | ✔ | ✘ ([1]) | ✔ |
| Raspberry Pi 2 | ✔ | ✘ ([1]) | [2] | 
| Raspberry Pi 3 ([3]) | ✔ | ✔ ([4]) | ✔ | 
| Raspberry Pi 4 ([3]) | ✔ | ✔ | ✔ |
